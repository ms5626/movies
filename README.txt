=====================
Movie Trailer Website
=====================

Movie Trailer Website is a simple Python app to show the user movie trailers. 
For each movie, users can click on the image of the poster and the corresponding trailer will be launched.


Quick start
-----------

1. To start the website from Python IDLE, copy all project files to a single folder.

2. Open entertainment_center.py using IDLE.

3. Run entertainment_center.py to generate and open the Website.

4. Once fresh_tomatoes.html is generated, double click the file to open the Website.
